Informationen zu den technischen Zeichnungen (STEP)
Information to the technical drawings (STEP)

Die Endung F1 steht stellvertretend für alle flanschkompatiblen Motorbaureihen:
Flanschgröße F1: AM811x, AM801x, AM311x, AM301x

The ending F1 representative all flange compatible motor series:
Flange code F1: AM811x, AM801x, AM311x, AM301x

Es wurden nur die Zeichnungen mit der Passfeder generiert. Wenn die Passfeder nicht benötigt
wird, kann sie einfach aus der Zeichnung entfernt werden.

Drawings are only with keyway-option. If you don’t need the keyway you can
delete it from the drawing.
